CSS

CSS can be written in 3 ways:

  1. Inline
  2. Internal
  3. External

Inline is written inline of the codes. Look comment for example(CTRL+U)

Internal CSS can be used for single HTML page. Internal CSS is written between "head" section with Style element. 

    Look comment for example:

External CSS is written in another sheet and must not contain any html element and must be saved with .css extention
To use an external style sheet, add a link to it in the "head" section of each HTML page
With an external style sheet, you can change the look of an entire web site, by changing one file!

  • CSS border

"Border" propety defines a border around an HTML element.
Tip: You can define a border for nearly all HTML elements.
Border uses padding and margin properities
Padding defines space between text and the border
Margin defines space outside of the border
